### 作业完成步骤

1. 注册并登录github
1. fork https://github.com/CubeADGroup/python-training 到你自己的账号下
1. git clone 你fork的repo到你的电脑上
1. 在homework下创建你自己名字的目录, 确保你的所有改动都在此目录下进行
1. edit, commit, push
1. 创建github PR
1. smite review PR
1. 如果有问题继续修改; 没问题merge
1. done
